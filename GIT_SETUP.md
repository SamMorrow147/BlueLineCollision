## Quick Push Commands

Run these commands to push your code:

```bash
# Stage all files
git add .

# Create initial commit
git commit -m "Initial commit - Blue Line Collision Center website"

# Push to GitHub
git push -u origin main
```

## After Pushing to GitHub

1. **Deploy to Vercel**:
   - Go to [vercel.com](https://vercel.com)
   - Click "Add New Project"
   - Import from GitHub: `SamMorrow147/BlueLineCollision`
   - Vercel will auto-detect Next.js
   - Click "Deploy"

2. **Your site will be live** at a Vercel URL (e.g., `blueline-collision-center.vercel.app`)

3. **Add Custom Domain** (optional):
   - In Vercel project settings → Domains
   - Add your domain (e.g., bluelinecollisioncenter.com)
