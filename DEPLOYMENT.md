# Vercel Deployment Guide

## Pre-Deployment Checklist

✅ **Build Test**: Project builds successfully (`npm run build`)
✅ **SEO**: All metadata, structured data, sitemap, and robots.txt configured
✅ **Configuration**: Next.js config optimized for production
✅ **Assets**: All images and static files in `public/blueline/` directory

## Quick Deploy to Vercel

### Method 1: Vercel Dashboard (Recommended)

1. **Push to Git**:
   ```bash
   git add .
   git commit -m "Initial commit - Ready for deployment"
   git push origin main
   ```

2. **Deploy on Vercel**:
   - Go to [vercel.com](https://vercel.com)
   - Sign in with GitHub/GitLab/Bitbucket
   - Click "Add New Project"
   - Import your repository
   - Vercel will auto-detect Next.js settings
   - Click "Deploy"

3. **Configure Domain** (Optional):
   - After deployment, go to Project Settings → Domains
   - Add your custom domain (e.g., bluelinecollisioncenter.com)
   - Follow DNS configuration instructions

### Method 2: Vercel CLI

1. **Install Vercel CLI**:
   ```bash
   npm i -g vercel
   ```

2. **Login**:
   ```bash
   vercel login
   ```

3. **Deploy**:
   ```bash
   vercel
   ```

4. **Deploy to Production**:
   ```bash
   vercel --prod
   ```

## Post-Deployment

1. **Verify Build**: Check that all pages load correctly
2. **Test Forms**: Ensure contact form works
3. **Check Images**: Verify all images load properly
4. **SEO Check**: Test sitemap at `/sitemap.xml` and robots.txt at `/robots.txt`
5. **Performance**: Run Lighthouse audit

## Environment Variables

No environment variables required for basic deployment.

## Build Settings (Auto-detected by Vercel)

- **Framework Preset**: Next.js
- **Build Command**: `npm run build`
- **Output Directory**: `.next`
- **Install Command**: `npm install`
- **Node Version**: 18.x (auto-detected)

## Important Notes

- All images in `public/blueline/` are committed to the repository
- The scraper script (`scripts/scrape.ts`) is excluded from build
- Typekit fonts load from external CDN (no additional config needed)
- Sitemap and robots.txt are auto-generated by Next.js

## Troubleshooting

If build fails:
1. Check Node.js version (should be 18+)
2. Run `npm install` to ensure dependencies are up to date
3. Run `npm run build` locally to identify issues
4. Check Vercel build logs for specific errors
